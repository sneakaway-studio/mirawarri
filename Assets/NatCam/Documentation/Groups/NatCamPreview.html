NatCamPreview

class in NatCamU.Core.UI / inherits from MonoBehaviour

Description

This is a helper component for properly displaying a texture on a UI panel. It is especially useful for displaying photos returned from NatCam.CapturePhoto() which may have different viewing orientations. It is also used for scaling the UI panel for the camera preview texture (to avoid stretching).



Public Functions

Apply Apply a texture with orientation and scale mode to the UI panel.

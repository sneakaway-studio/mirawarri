NatCamFocuser

class in NatCamU.Core.UI / inherits from MonoBehaviour

Description

This is a helper component for detecting focus gestures to focus the active camera. It can be applied to UI panels. Note that the camera's focus mode must have the FocusMode.TapToFocus bit set.



Public Functions

StartTracking Start tracking focus gestures.
StopTracking Stop tracking focus gestures.


Public Variables

IsTracking Are focus gestures being tracked?

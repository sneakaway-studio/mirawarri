Utilities.RotateImage

public static void RotateImage (Texture2D texture, Orientation rotation, IDispatch workerDispatch, IDispatch mainDispatch, PhotoCallback callback);

Description

Rotate a texture.

This will rotate a texture in place using the Color32 buffer overload. This process is very expensive, so it is best dispatched off to a worker thread. To do so, create IDispatches and pass them in. The 'workerDispatch' is where the actual rotation will occur, and the 'mainDispatch' is where the results will be enqueued to be used to update the texture.
 

IDispatch main = new MainDispatch(), worker = new ConcurrentDispatch();
Utilities.RotateImage(photo, orientation, worker, main, (rotated, unused) => {
        // Do stuff with `rotated`
        Texture2D.Destroy(rotated); // Remember to free memory
        main.Release(); worker.Release();
});

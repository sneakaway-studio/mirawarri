Utilities.RotateImage

public static void RotateImage (Color32[] dst, Color32[] src, int width, int height, byte orientation);

Description

Rotate an image buffer.

The orientation parameter is equivalent to the Orientation struct.
 

Color32[] src = texture.GetPixels32(), dst = new Color32[src.length];
Orientation orientation = Orientation.Rotation_90 | Orientation.Mirror;
Utilities.RotateImage(dst, src, texture.width, texture.height, (byte)orientation);

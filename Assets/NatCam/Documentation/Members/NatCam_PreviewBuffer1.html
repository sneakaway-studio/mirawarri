NatCam.PreviewBuffer

public static bool PreviewBuffer (Color32&[] colors, Int32& width, Int32& height);

Description

Get the preview buffer for the current frame. This overload copies the pixel data from native memory to a managed Color32 array. It is worse for memory management as the CLR lacks deterministic memory management, so only use it when you absolutely need to.
